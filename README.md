# Niamnns - Premium Custom Linktree

A highly optimized, minimal, and premium custom Linktree alternative built for Software Engineers and UI/UX Designers. 

## Features
- **Tech Premium Light Mode**: Apple-inspired clean aesthetic.
- **Glassmorphism UI**: Frosted glass cards with ultra-smooth hover animations.
- **Ambient Glow**: Subtle Mesh Gradient background effects (Sky Blue & Royal Blue).
- **SEO & A11y Ready**: Fully equipped with Open Graph tags and keyboard navigation support (`:focus-visible`).
- **Zero Dependencies**: Built entirely with Vanilla HTML, CSS, and JS for maximum speed and performance.
